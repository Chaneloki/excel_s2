# Map Walker 美術資產

這個資料夾放 `03_map_walker_hotspot_demo` 零件使用的環境插畫。

| 檔案 | 用途 |
|---|---|
| `bg_exhibition_hall_main_d1.png` | D1主展場場景背景，來源是 `3case/case1_bg/d1_exhibition_hall_main.png`，複製進來方便Godot專案內引用，原始檔仍以`3case/case1_bg`為主。 |

熱點icon（放大鏡圖示）目前用程式碼動態繪製（見`03_map_walker_hotspot_demo.gd`的`_create_magnifier_texture()`），
沒有對應的圖片資產，等正式美術定案後再換成PNG素材。
